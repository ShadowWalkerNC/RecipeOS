# 🍳 RecipeOS

> **The professional kitchen — for everyone.** From first-time cooks to seasoned chefs, RecipeOS is the all-in-one kitchen operating system built for how real kitchens work.

---

## Vision

RecipeOS bridges the gap between home cooking and professional kitchen workflows. Whether you're following your first recipe or running mise en place for 200 covers, RecipeOS gives you the tools that pros use — with a UX that welcomes beginners.

---

## Feature Overview

### 📖 Recipe Management
- Create, edit, and browse recipes at any skill level (Beginner → Advanced)
- Step-by-step instructions with skill-level tags
- **Scan a Recipe** — use your camera to scan a physical recipe card or printed page and import it automatically via OCR + AI parsing
- AI-powered recipe generation from a list of ingredients you already have

### 🥕 Ingredient & Inventory
- Full pantry/inventory tracker linked to your recipes
- Input ingredients you have → get recipe ideas generated by AI
- Automatic "what's missing" detection when you pick a recipe
- Grocery list auto-generated from recipe needs vs. current stock

### ⚖️ Unit Conversion & Scaling
- Professional-grade conversion: weight ↔ volume (g, oz, kg, lb, ml, fl oz, cups, tbsp, tsp, liters)
- Temperature: °C ↔ °F ↔ °Gas Mark
- Scale any recipe by servings, batch size, or yield %
- Density-aware conversions (e.g. "1 cup of flour" → grams using ingredient-specific density)

### 📋 Prep Lists (Professional Kitchen Workflow)
- Build **mise en place prep lists** like a professional kitchen
- Tasks include: wash, peel, chop, weigh, portion, marinate, etc.
- Assign prep tasks to time blocks or team members
- Pre-service checklist mode with strike-through and progress tracking
- Templates for common prep workflows (e.g. "Soup Day", "Baking Prep", "Protein Butchery")

### 🤖 AI Integration (Gemini)
- Recipe generation from available ingredients
- Scan & import recipes from physical cards or images (OCR + AI parse)
- Smart substitution suggestions
- Skill-level adaptation: simplify or elevate any recipe

## Getting Started

### Prerequisites
- [Android Studio](https://developer.android.com/studio) (latest stable)
- A [Gemini API key](https://aistudio.google.com/apikey)

### Setup
```bash
# 1. Clone the repo
git clone https://github.com/ShadowWalkerNC/RecipeOS.git
cd RecipeOS

# 2. Create your env file
cp .env.example .env
# → Add GEMINI_API_KEY to .env

# 3. Open in Android Studio → Run on emulator or device
```

> **Note:** Remove `signingConfig = signingConfigs.getByName("debugConfig")` from `app/build.gradle.kts` before running.
